3. Design brief foundation

If you choose the recommended route, these become the first design tokens for Flutter.

Recommended master direction

Cinematic Emerald Luxury: dark gradient background, glass cards, emerald trust/growth accent, amber opportunity CTA, violet/cyan reserved for AI moments.

  • Dark mode first, but maintain an accessible light fallback later.
  • Use glass only for elevated/featured content, not every component.
  • Recruiter screens should reduce glow and prioritize data clarity.

5. Spacing and motion system

Use an 8dp rhythm for Flutter. Motion follows Material principles: continuity, container transform, fade-through, shared element hero.

Spacing scale

xs4dp, tiny alignment
sm8dp, icon/text gaps
md16dp, card padding
lg24dp, section spacing
xl32dp, screen rhythm
2xl48dp, hero separation

Motion tokens

fast150ms, press feedback
base240ms, chips/cards
route320-400ms, page transition
stagger30-45ms per list item
curveemphasized decelerate / spring
reduceMotiondisable parallax, shimmer, stagger
Shared element heroJob card expands into detail page. Keeps company logo/title spatially continuous.
Apply micro-interactionButton compresses slightly, glows, then confirms with check state and haptic.
Staggered feed revealCards enter 30-45ms apart using opacity + translateY only.
Cinematic depthSubtle blur and parallax on hero layers only, never core text.
