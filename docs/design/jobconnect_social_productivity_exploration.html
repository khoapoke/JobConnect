Color direction decision

Blue, violet, or hybrid?

All options keep the same Modern Social Productivity structure. The question is emotional tone: trust, AI magic, or balanced cinematic productivity.

Locked: Blue × Violet Hybrid
A · Cinematic Blue

Professional, clean, recruitment-safe

Risk: may become generic SaaS/corporate unless cinematic depth is strong.

C · Blue × Violet Hybrid

Social productivity with cinematic AI

Blue handles trust, tabs, search, and primary actions. Violet handles AI insights and magic moments. Cyan is a rare glow/focus color.

My recommendation: strongest match for Instagram × Linear × Raycast.

Design brief snapshot

The system this preview implies.

If this direction feels right, we can convert it into a full design brief, Flutter theme tokens, and a real job feed implementation plan.

Style thesis

  • Modern Social Productivity Design: social feed behavior, professional hiring content.
  • Cinematic Minimalism: depth, gradients, glass, but restrained content hierarchy.
  • Linear clarity: sharp typography, whitespace, subtle borders, low visual noise.
  • Raycast speed: instant search, quick actions, responsive feedback.

Light mode companion

Light mode should be clean, not plain

Use off-white, blue/violet tint, soft cards, and fewer glow effects. Keep the same feed/search model so the product identity survives theme changes.

Rules to avoid noise

  • Only one primary blue CTA per screen.
  • Violet is reserved for AI/recommendation moments.
  • Cyan glow is rare: search focus, active command, or hero shimmer.
  • Cards use subtle lift, not heavy skeuomorphic shadows.
  • Feed stays readable before it becomes cinematic.
